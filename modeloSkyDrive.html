Conoce los diferentes modelos de autos SkyRide





SkyDrive 4.0

Basado en el primer modelo lanzado en el año 2025. Este es el medio de transporte preferido para los ciudadanos de todo el mundo.

SkyDrive 5.0

Este modelo fue diseñado para ser más rápido y eficiente que su predecesor. Sin embargo cuenta con más opciones de personalización. Una pieza unica.

SkyDrive MK VII

Modelo basado en sus especificaciones de alto rendimiento preparado para largos viajes. Cuenta con un sistema de ahorro y eficiencia sin precedentes.

SkyDrive SD-XX

Nuestro modelo multi-propostio cuenta con capacidad para viajar en cualquier situación y sobre cualquier clima. Su sistema de calentamiento integrado permite viajar sobre los 15000 Metros.



Alef Model A

Este modelo es 100% eléctrico, tiene capacidad para uno o dos ocupantes y tiene un rango de vuelo de 177 Kilometros.

Alef Model B

Este modelo al igual que su anterior versión es 100% eléctrico, cuenta con más capacidad y tiene un rango de vuelo de 300 Kilometros.

Terrafugia TF-X

Un modelo de automóvil volador que se espera que sea completamente autónomo. Este modelo está diseñado para ser más seguro y fácil de usar que cualquier otro modelo de automóvil volador en el mercado.

PAL-V Liberty

Un modelo de automóvil volador que se convierte en un helicóptero. Este modelo tiene capacidad para dos ocupantes y tiene un rango de vuelo de 500 Kilometros.
